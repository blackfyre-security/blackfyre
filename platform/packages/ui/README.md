# @blackfyre/ui

Shared React component library for BLACKFYRE admin and portal frontends.

## Components

Badge, Button, Card, ComplianceStepper, DataTable, EmptyState, ErrorBoundary, FindingCard, FrameworkScoreCard, Input, and more.

## Usage

```tsx
import { Button, Card, DataTable } from "@blackfyre/ui";
```

Built with React 18, Framer Motion, Lucide icons, and the shared Tailwind preset.
